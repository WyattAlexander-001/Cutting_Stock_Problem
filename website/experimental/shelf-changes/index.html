Shelf-pack Bin Packing Visualization

Upload JSON file containing items...

Or manually add items


Add your items manually by clicking "Add Item" and entering the measurements of the desired piece. Once all desired pieces are added, click "Load Items" to visualize cuts.

If using a JSON file loaded with width, height, and quantity for the pieces, cuts will be visualized upon uploading the JSON file. JSON input is standardized between Guillotine and Shelf-Pack algorithms, so JSON files generated by Guillotine can be used for Shelf-Pack, and vice versa.
